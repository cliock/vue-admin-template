### 组件说明
- 定制的表单样式组件node
### 参数参考
##### Attributes
 参数 | 说明 | 类型 | 可选值 | 默认值
 -----|-----|-----|------|-----
type | 表单类型 | String | input类型 | text
disabled | 是否禁用 | Boolean | true、false | false
echoValue | 回显数据,用于详情表单数据回显 | String | 无 | 无

##### select类型
 参数 | 说明 | 类型 | 可选值 | 默认值
 -----|-----|-----|------|-----
options | select类型的下拉选项,需要type为select | Array | 无 | []

##### data数据
 参数 | 说明 | 类型 | 可选值 | 默认值
 -----|-----|-----|------|-----
 value | 表单输入的数据 | String | 无 | 无
